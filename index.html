Frequently Asked Questions

Everything you need to know

How do I install these scripts?

‚ñº

Installing scripts is simple. First, install a userscript manager extension like Tampermonkey or Violentmonkey in your browser. Then, click on any script link from our catalog and the manager will prompt you to install it. Once installed, the script will automatically run when you open Discord in your browser.

Are these scripts safe to use?

‚ñº

All scripts in our catalog are thoroughly tested and designed with security in mind. They only modify your local Discord experience and don't access or transmit any sensitive data. However, always review scripts before installation and only use trusted userscript managers.

Can I customize the scripts?

‚ñº

Yes! Each script comes with customizable settings that you can adjust directly in your userscript manager. You can modify timings, keywords, colors, and other parameters to match your preferences without any coding knowledge.

Do I need coding knowledge?

‚ñº

Not at all! Our scripts are designed for everyone. Simply install them through your userscript manager and adjust the settings through user-friendly interfaces. No programming experience required.

Will these work on mobile?

‚ñº

These scripts are designed for desktop browsers where userscript managers are supported. Mobile browsers have limited support for userscripts, so we recommend using these tools on your desktop or laptop for the best experience.
